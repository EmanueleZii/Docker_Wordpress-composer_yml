### Docker Wordpress/PhpMyAdmin/MariaDb

## Istruzioni Per L'uso

- Copiare il file docker-composer.yml su una directory / cartella
- Lanciare il comando da terminale nella directory in cui ce il file docker-composer.yml
  '''
  docker-compose up -d
  '''
## Apri WordPress / phpMyAdmin
- WordPress → http://localhost:8080
- phpMyAdmin → http://localhost:8081
